Bearer Tokenの取得方法
1. X Developer Portal でアプリを作成
2. App → Keys and Tokens → Bearer Token をコピー
⚠ OAuth 2.0 User Access Token。通報APIには users.report スコープが必要です。
User Access Tokenの取得方法
1. Developer Portal → User authentication settings → OAuth 2.0 を有効化
2. Scopes: tweet.read users.read users.report
3. PKCE フローでアクセストークンを取得
